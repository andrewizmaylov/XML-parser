# XML процессор

Пакет позволяет организовать импорт данных из внешнего XML файла.
Данные импортируются по предоставленной маске.

Большие файлы читаются и записываются в БД чанками, обеспечивая устойчивость приложения и контроль потребляемой памяти.

Пакет позволяет производить 'докачку' данных при обрыве соединения либо ручной остановке.

## Требования

- PHP 8.4

## Установка

```sh

composer require andrewizmaylov/XML-parser

```
